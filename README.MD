# ATM Simulator (Python)

A simple ATM simulator built for my Python course capstone. It demonstrates user authentication, deposits/withdrawals, balance inquiry, and PIN changes. This repository includes code and documentation.

## Features
- Login with username + 4-digit PIN
- View balance
- Deposit & withdraw funds with validation
- Change PIN
- Menu returns after each action until Exit

## How to Run
1. Install Python 3
2. Download this repository
3. Run the script:
   ```bash
   python ATM.py
